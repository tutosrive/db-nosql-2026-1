## 📖 Acerca del Proyecto

Este repositorio funciona como una biblioteca de estudio para Bases de Datos No Relacionales. Contiene material de teoría, prácticas de laboratorio, ejercicios de clase y presentaciones para comprender modelos NoSQL, consultas y casos de uso.

## 📂 Estructura del Repositorio

El contenido está organizado para facilitar la revisión y el trabajo por temas:

- 📁 **[`Ejercicios/`](https://github.com/tutosrive/db-nosql-2026-1/tree/main/Ejercicios)**: Actividades y desafíos prácticos sobre bases de datos NoSQL.
- 📁 **[`PIAA/`](https://github.com/tutosrive/db-nosql-2026-1/tree/main/PIAA)**: Material de presentaciones, informes y ejercicios de PIAA.
- 📁 **[`Parciales/parcial1/`](https://github.com/tutosrive/db-nosql-2026-1/tree/main/Parciales/parcial1)**: Recursos y evaluaciones del primer parcial.
- 📁 **[`Presentaciones/`](https://github.com/tutosrive/db-nosql-2026-1/tree/main/Presentaciones)**: Diapositivas y material visual de clase.
- 📁 **[`Talleres/`](https://github.com/tutosrive/db-nosql-2026-1/tree/main/Talleres)**: Talleres guiados para aplicar conceptos en ejemplos reales.
- 📁 **[`notas-clase/`](https://github.com/tutosrive/db-nosql-2026-1/tree/main/notas-clase)**: Apuntes y resúmenes teóricos de las sesiones.

## 🛠️ Tecnologías y Herramientas

El material del repositorio está enfocado en las siguientes tecnologías:

* **Conceptos:** Modelos NoSQL, documentos, clave-valor, grafos y bases de datos distribuidas.
* **Lenguajes:** MongoDB Shell / Compass, JSON, Python (para notebooks y scripts), SQL comparativo.
* **Entornos:** Jupyter Notebook / Google Colab.
* **Herramientas:** MongoDB, CouchDB, Redis, y otras plataformas NoSQL según se aborde en cada actividad.

## 🚀 Cómo usar este repositorio

Puedes trabajar con el contenido del repositorio de las siguientes maneras:

### 1. Consultar los apuntes y presentaciones

Revisa las carpetas de `notas-clase`, `Presentaciones` y `PIAA` para encontrar teoría, ejemplos y explicaciones de los temas vistos en clase.

### 2. Resolver ejercicios y talleres

Accede a `Ejercicios`, `Talleres` y `Parciales/parcial1` para practicar con problemas guiados, soluciones y casos prácticos.

### 3. Usar archivos en Google Colab o Jupyter

Si tienes archivos `.ipynb`, puedes:

1. Abrirlos en [Google Colab](https://colab.research.google.com/) para ejecutarlos sin instalaciones locales.
2. O trabajar en local con Jupyter si prefieres:
   ```bash
   git clone https://github.com/tutosrive/db-nosql-2026-1.git
   cd db-nosql-2026-1
   jupyter notebook
   ```
